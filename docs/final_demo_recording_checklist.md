# Final Demo Recording Checklist

## Goal

Record a clear 3-minute demo showing QuotePilot AI working as a Track 4 Autopilot Agent.

## Recording structure

### 0:00–0:20 — Problem

Say:

"Businesses receive messy quote requests by email or chat. Turning those requests into accurate quotes requires reading the message, checking products, checking stock, calculating price, reviewing risk, and drafting a reply. This is repetitive but risky to automate without human approval."

### 0:20–0:45 — Solution

Say:

"QuotePilot AI is a Qwen-powered Autopilot Agent that automates the RFQ-to-Quote workflow while keeping humans in control of risky actions."

### 0:45–1:20 — Clean inquiry demo

Show:

- Choose Clean inquiry.
- Click Analyze with Qwen + Run Business Tools.
- Show Qwen Extraction tab.
- Show Tool Calls tab.
- Show Quote Draft tab.

### 1:20–1:55 — Human approval + PDF

Show:

- Human Approval tab.
- Select Approve draft.
- Open PDF + OSS tab.
- Generate quote PDF.
- Show Download quote PDF.

### 1:55–2:25 — Ambiguous inquiry demo

Show:

- Choose Ambiguous inquiry.
- Run workflow.
- Show missing fields / clarification.

### 2:25–2:50 — Risky inquiry demo

Show:

- Choose Risky inquiry.
- Show risk flags / approval required.
- Explain email is not sent automatically.

### 2:50–3:00 — Closing

Say:

"QuotePilot demonstrates a production-minded Autopilot Agent: Qwen reasoning, deterministic tools, human approval, and business-ready PDF output. Alibaba OSS and deployment support are implemented and pending live cloud activation."

## Must show in video

- Qwen extraction output
- Tool calls
- Policy guard
- Human approval
- PDF generation
- No auto-send behavior

## Do not show

- `.env`
- API keys
- Alibaba AccessKeys
- Identity documents
- Payment details
